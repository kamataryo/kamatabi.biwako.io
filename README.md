## Required

- Node.js >= 4

## Development

```shell
$ git clone git@github.com:kamataryo/kamatabi.biwako.io.git
$ cd kamatabi.biwako.io
$ npm install
```

Start a dev-http-server:

```shell
$ npm start
```

Then build:

```shell
$ npm run build
```

## Automated testing

```shell
$ npm test
```
